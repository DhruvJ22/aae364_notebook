## Environment Installation

```bash
git clone git@github.com:jgoppert/aae364_notebook ~/aae364_notebook
```

Create your environment from the environment.yml file.

```bash
cd ~/aae364_notebook
conda env create -f environment.yml
```

## Start JupyterLab

```bash
. activate aae364
cd ~/aae364_notebook
jupyter lab
```
